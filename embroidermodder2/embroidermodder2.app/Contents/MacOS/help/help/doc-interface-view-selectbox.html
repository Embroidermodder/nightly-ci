Selection Box
The selection box is used for selecting and deselecting objects in the view. There are two types of modes that can be used:
  • PickAdd Mode
    Repeated use adds objects to the selection. (default mode)

  • PickNew Mode
    Repeated use deselects any previously selected objects, selecting only the new objects.
Directional selection is also provided for more advanced selecting of objects. It is not a mode and cannot be toggled off or disabled.
PickAdd Mode
This is the default mode of the selection box.
  1. When left clicking the mouse in the view, the selection box will appear.
  2. Moving the mouse will resize the selection box.
  3. By left clicking the mouse a second time, the selection box will select objects and then the selection box will disappear.
  4. To add more objects to the selection, repeat this procedure.
To deselect indivual or multiple objects, hold down shift while making the selection box.
To deselect all objects press the Escape Key.

PickNew Mode
To switch to this mode, click the PickAdd button in the top-right corner of the property editor. Clicking the button again will switch back to PickAdd mode.
  1. When left clicking the mouse in the view, the selection box will appear.
  2. Moving the mouse will resize the selection box.
  3. By left clicking the mouse a second time, the selection box will deselect any previously selected objects, selecting only the new objects and then the selection box will disappear.
  4. To add more objects to the selection, repeat this procedure.
To add more objects to the selection while in PickNew mode, hold down shift while making the selection box for unselected objects.
To deselect previously selected objects, hold down shift and select those objects. To deselect all objects press the Escape Key.

Directional Selection
When the selection box is active, moving the box to the right or left of the inital pick point will change the color of the selection box. This change in color is significant and the selection box operates differently depending on the direction.
  • If the x-coordinate of the second point picked is to the right of the first pick point, it will select any objects contained inside the box. This is called a Window selection.
  • If the x-coordinate of the second point picked is to the left of the first pick point, it will select any objects contained inside the box as well as any objects that are intersecting the box. This is called a Crossing selection.
