Toolbars
The toolbar area is a convenient way to access commands. All commands found in the toolbar can also be found in the menus or by typing the command name into the command prompt.
Showing and Hiding Toolbars
To show or hide various toolbars, you can right click in the menu bar area or in the toolbar areas and a context menu will appear. Check or uncheck the toolbar names listed in the context menu to show and hide them accordingly.
