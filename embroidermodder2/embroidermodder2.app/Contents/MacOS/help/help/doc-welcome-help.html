Accessing Help
There are multiple ways of accessing the help file and other useful documentation:
  • Help Menu > Help will open the documentation index.
  • Clicking on the question mark/help icon in the toolbar area will open the documentation index.
  • Typing 'HELP' into the command prompt will open the documentation index.
  • Pressing the F1 Key.
  • Hover the mouse pointer over any toolbar icon and information about it will display in the status bar.
  • Help Menu > Tip of the Day contains various bits of information about the application and how to use it more effectively.
  • The latest documentation can always be found online.
