Embroidermodder Documentation
Welcome
Welcome to the Embroidermodder 2 Documentation. It contains detailed information on how to use the software and various techniques that will save you time and help you to create embroidery designs and vector graphics.
Contribute
Please bear with us as the documentation isn't fully complete and may be missing sections. The documentation constantly improving and the latest documentation is available online. Documentation can sometimes be a less polished area when it comes to open source projects. It is not a highly technical task to contribute documentation, which is why we encourage the community to contribute. If you would like to help improve the documentation, please contact us. For those of you who are GitHub saavy, please open a pull request.
