Grid
The grid is shown in the background of the view and can be toggled on or off. To toggle the grid on or off, click the 'GRID' button that appears in the status bar at the bottom of the screen.
  • When the 'GRID' button is pressed in, the grid will become visible in the view.
  • When the 'GRID' button is unpressed, the grid will become hidden in the view.
A common use of the grid, would be to set it to the same size as the hoop you will be using.
An optional origin point can be shown while the grid is visible. It is always located at point (0,0).
Types of Grids
  • Rectangular Grid
  • Circular Grid
  • Isometric Grid
Grid Settings
Settings for the grid can be found in the Settings Menu > Settings > Grid/Ruler. By right-clicking on the 'GRID' button, you can quickly access grid settings.

Grid geometry and type can be changed in the settings menu.
The origin can be turned on or off in the settings menu.
