System Requirements
Embroidermodder 2 can run on the Raspberry Pi Model B. That should give you a basic idea of a minimal system, but we have listed several other individual requirements below:
  • Ubuntu, Debian, Windows or Mac OS X
  • 1GB of available hard-drive space for installation (solid state drive recommended)
  • 700MHz 32-bit or 64-bit processor (Intel Core i3/i5/i7 or higher recommended)
  • 512GB of RAM (4GB or higher recommended)
  • 1024x768 display (1920x1080 or higher recommended)
  • 3D capable graphics card (Nvidia Geforce 8 Series or higher recommended)
  • Mouse (Wheel mouse with side buttons recommended)
  • Keyboard (full size with keypad recommended)
  • An Internet connection for downloading the software (Not required for installation)
